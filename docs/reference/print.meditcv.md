# Print a meditcv object

Print a meditcv object

## Usage

``` r
# S3 method for class 'meditcv'
print(x, ...)
```

## Arguments

- x:

  A `meditcv` object from
  [`sensitivity_meditcv()`](https://causalfragility-lab.github.io/RobustMediate/reference/sensitivity_meditcv.md).

- ...:

  Ignored.

## Value

The input object, invisibly. Called for its side effect of printing a
formatted medITCV report to the console.
